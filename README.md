🔐 AI Password Strength Checker

🚀 Live Demo
👉 https://password-ai-vzw8.vercel.app

An AI-powered web application that analyzes password strength using Machine Learning and provides instant feedback on how secure a password is.

This project combines a React frontend, Flask backend, and a Scikit-learn ML model, fully deployed on the cloud.

✨ Features

-AI-based password strength detection

-Real-time password analysis

-REST API for password prediction

-Fully deployed full-stack application

-Responsive and simple UI

🧠 How It Works

The application analyzes several features of a password:

-Password length

-Presence of uppercase characters

-Presence of lowercase characters

-Presence of numbers

-Presence of special characters

-Number of digits

These features are passed to a Machine Learning model trained using Scikit-learn, which predicts whether the password is Weak, Medium, or Strong.
